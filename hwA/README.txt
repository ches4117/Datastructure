This program has two functiom, read file and save file having another name.

It has a start menu, and we can choose what we want to do.

Choose one, read data. Entering the filename, and the program will save the data into an array of struct.

Choose two, show data. We can know what is in your file by showing on screen.

Choose three, write data. Input the filename you want to save, and we can have another file.

choose zero, exit the program. You can exit this program.
